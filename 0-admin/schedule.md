# 🗓 Calendrier Détaillé du Bootcamp

Ce calendrier définit la progression hebdomadaire pour maîtriser le développement Full-stack.

## 🟦 PHASE 1 : Fondations Web (3 Semaines)
**Objectif :** Maîtriser l'UI et la logique de programmation.

### Semaine 1 : HTML5 & CSS3 Moderne
- Structure sémantique et accessibilité (A11y).
- Mise en page avancée : **Flexbox** et **CSS Grid**.
- Design Responsif : Media Queries et introduction à **Tailwind CSS**.

### Semaine 2 : JavaScript ES6+
- Variables (`let`/`const`), fonctions fléchées, et déstructuration.
- Manipulation du **DOM** et gestion des événements.
- Programmation asynchrone : Promises, `async/await` et **Fetch API**.
- Creation de 2 jeux (Tic Tac Toe , Quiz Game)

### Semaine 3 : Git & Workflow Professionnel
- Gestion de version : Branching, Merging et Pull Requests sur GitHub.
- **Projet :** Création d'un portfolio dynamique hébergé sur Vercel.

---

## 🟨 PHASE 2 : L'Écosystème React (2 Semaines)
**Objectif :** Construire des interfaces modulaires et performantes.

### Semaine 4 : Fondamentaux de React
- Installation avec Vite et syntaxe JSX.
- Architecture : Composants, **Props** et rendu conditionnel.
- Hooks de base : `useState` et `useEffect`.
- Projet : Todo list

### Semaine 5 : React Avancé
- Navigation : **React Router** pour les Single Page Apps (SPA).
- État Global : Context API et introduction à **Zustand**.
- **3 Projets :** Calculatrice de budget, Application Météo, et (Api app,...)

---

## 🟩 PHASE 3 : Backend & Base de Données (2 Semaines)
**Objectif :** Créer des serveurs sécurisés et persistants.

### Semaine 6 : Node.js & Express
- Architecture de serveur et routage REST API.
- **Middlewares** : Logging, sécurité et gestion des erreurs.
- Sécurité : Authentification via **JWT** et hachage **Bcrypt**.

### Semaine 7 : MongoDB & Mongoose
- Modélisation NoSQL : Schémas, validation et relations (**Populate**).
- Opérations CRUD avancées.
- **Projet :** API de gestion de blog avec authentification.

---

## 🚀 PHASE 4 : Intégration MERN (1 Semaine)

### Semaine 8 : Projet MERN Full-stack
- Connexion du Frontend au Backend.
- Gestion de l'état d'authentification (Persistance utilisateur).
- **2 Projets :** Plateforme de discussion en temps réel et Task Manager complet.

---

## 📱 PHASE 5 : Développement Mobile (5 Semaines)
**Objectif :** Créer des applications natives performantes.

### Semaine 9-10 : Intro à React Native
- Configuration de l'environnement avec **Expo**.
- Composants natifs : `View`, `Text`, `FlatList`, `Image`.
- Navigation Mobile : Stack & Tab Navigation.

### Semaine 11-12 : Mobile Avancé
- APIs natives : Caméra, Galerie, Géolocalisation.
- Stockage local : **AsyncStorage**.
- UI : **Animations** et **Notifications Push**.
- **2 Projets :** App de scan QR et Tracker de Fitness.

### Semaine 13-14 : Projet Mobile Final
- Développement d'une application mobile complète connectée au backend MERN existant.
- **Build & Publication** : Préparation au déploiement via Expo EAS.
