# Syllabus Officiel : Bootcamp Fullstack MERN & Mobile

**Objectif du cours :** Permettre aux étudiants d'acquérir les compétences nécessaires pour concevoir, sécuriser et déployer des applications web et mobiles modernes à partir de zéro.

---

### Module 1 : Fondations du Web
* **HTML & CSS** : HTML sémantique, Flexbox, Grid et Design Responsif.
* **JavaScript** : Syntaxe ES6+, manipulation du DOM et API Fetch.
* **Gestion de version** : Flux de travail Git professionnel et collaboration GitHub.
* **Projets** : Page d'atterrissage statique, Mini-Portfolio et applications JS logiques.

### Module 2 & 3 : Frontend Moderne (React)
* **Les Bases** : JSX, Composants, Props et gestion d'État (State).
* **Hooks** : `useState`, `useEffect`, `useMemo` et `useCallback`.
* **Patterns Avancés** : Context API, Hooks personnalisés et Routage côté client.
* **Gestion d'État** : Introduction à Redux Toolkit et Zustand.
* **Projets** : Gestionnaire de tâches (To-Do), Tableau de bord (Dashboard), Flux d'authentification Frontend.

### Module 4 & 5 : Backend & Base de données (Node.js & MongoDB)
* **Logique Serveur** : Bases de Node.js et middleware Express.js.
* **Sécurité** : Authentification via JSON Web Tokens (JWT) et hachage de mots de passe.
* **Modélisation de données** : Schémas MongoDB, opérations CRUD et relations NoSQL.
* **Projets** : API RESTful sécurisée, application pilotée par base de données.

### Module 6 : Intégration Fullstack (MERN)
* Connexion du frontend React au backend Node/Express.
* Communication en temps réel (WebSockets/Socket.io).
* **Projet Final MERN** : Application web complète avec Auth, CRUD et recherche.

### Module 7, 8 & 9 : Développement Mobile (React Native)
* **Configuration Mobile** : Environnement iOS et Android.
* **Composants Noyaux** : View, Text, ScrollView et FlatList.
* **Navigation** : Modèles de navigation par pile (Stack) et par onglets (Tabs).
* **Fonctionnalités Avancées** : Notifications Push, stockage local et animations.
* **Projet Final** : Application mobile complète et fonctionnelle.

---

## 🎓 Conditions d'obtention du certificat
1. Finalisation de tous les projets spécifiques aux modules.
2. Déploiement réussi du Projet Final Web MERN.
3. Démonstration réussie de l'Application Mobile finale.
